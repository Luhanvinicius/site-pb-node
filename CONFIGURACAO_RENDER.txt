✅ CÓDIGO AJUSTADO PARA RENDER!

═══════════════════════════════════════════════

📋 CONFIGURAÇÃO NA TELA DO RENDER:
───────────────────────────────────────────────

1. NAME:
   tactical (ou deixe site-pb se preferir)

2. ROOT DIRECTORY:
   tactical
   ⚠️ Se seu repo tem pasta "tactical", coloque isso aqui!

3. BUILD COMMAND:
   npm install

4. START COMMAND:
   npm start

5. Clique em "Create Web Service"

═══════════════════════════════════════════════

🔐 DEPOIS DO DEPLOY - VARIÁVEIS DE AMBIENTE:
───────────────────────────────────────────────

1. Vá em "Environment" (menu lateral)

2. Adicione:
   DATABASE_URL = postgresql://postgres:1988@37.148.132.118:5432/postgres
   NODE_ENV = production

3. Save Changes

═══════════════════════════════════════════════

🔒 IMPORTANTE - FIREWALL DA VPS:
───────────────────────────────────────────────

Certifique-se que porta 5432 está aberta:
   sudo ufw allow 5432/tcp

E PostgreSQL aceita conexões externas.
